# How to run locally

## Install dependencies

```bash
npm install
```

## Create a .env.local file

```env
MONGODB_URI=<xxxxx>
MONGODB_DB=<xxxxx>
SECRET_COOKIE_PASSWORD=<xxxxxx>
```
note: the credentials sent to you by email

## Run the app

```bash
npm start
```
